# Crypto Recommendation Service - crypteacher

## Running directly

```sh
./mvnw spring-boot:run
```

## Creating Docker Image

```sh
docker build -t xm/crypteacher .
```

## Running Docker Container

```sh
docker run -p 8080:8080 xm/crypteacher
```
